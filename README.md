# SKKUNION - Frontend

# 소프트웨어공학개론 Team 1

# 📒 프로젝트 소개
React를 사용한 동아리 지원 밎 부원 모집 플랫폼(웹서비스) SKKUNION의 Frontend

# 👫 프론트엔드 팀 멤버
정현철, 최영주

# ⏳ 개발 기간
2023.09 - 2023.11

# 🔩 개발환경
* React.js
⠀
# 🪽주요 기능
* 회원 관련
  * 로그인
  * 회원가입
  * 회원정보수정
* 동아리 지원 관련
  * 동아리 정보 리스트 조회
  * 동아리 정보 단건조회
* 동아리 부원 모집 관련
  * 사용자 리스트 조회
  * 사용자 단건 조회
  * 동아리 정보 수정
* 채팅 관련
  * 채팅 전송
  * 채팅방 생성
  * 채팅방 정보 조회

# 🔨실행 방법
## 패키지 설치
`npm install` 

## 로컬에서 실행
`npm start`
